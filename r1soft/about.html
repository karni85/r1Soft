About

This is a simple Angular.js application that shows how to do the following things

  • How to integrate Angular.js with Require.js
  • How to write single page applications with Angular.js
  • How to write Angular.js controllers
  • How to write Angular.js services
  • How to write Angular.js directives
  • How to use Angular.js bindings
  • How to use Angular.js to provide live bound CSS values
  • How to use Angular.js with WebSockets
  • How to use Angular.js with LocalStorage
